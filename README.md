# Система анализа успеваемости студентов

Данная система представляет собой аналитическую платформу для мониторинга успеваемости студентов, разработанную на стеке **ASP.NET Core**.

## Основные характеристики
* **Технологический стек**: ASP.NET Core, PostgreSQL, Dapper.
* **Архитектура**: Реализован переход к принципам **Clean Architecture** с разделением на слои Application и Infrastructure.
* **Аналитика**: Включает **8 специализированных SQL-отчетов** для мониторинга активности и успеваемости студентов.
* **Безопасность**: Поддержка двух методов аутентификации:
    * **JWT (JSON Web Token)**
    * **Cookie-based аутентификация**
* **Тестирование**: Проект покрыт unit-тестами для обеспечения надежности кода.
* **CI/CD**: Настроен пайплайн в **GitHub Actions** для автоматизации сборки и тестирования проекта.

1. **Подготовка**: Убедитесь, что у вас установлена СУБД PostgreSQL.
2. **Конфигурация**: Настройте строку подключения к вашей базе данных в файле `appsettings.json`.
3. **Развертывание БД**: Примените миграции для создания структуры таблиц:
